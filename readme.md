> open-source engine with API for Real-time questions In Hebrew.

## Install

```sh
npm install hejs
```
## import 

```sh

import { Answers } from 'hejs'

```
## Usage

```sh

const { 
  direction, 
  meaning,  
  news, 
  translate, 
  weather 
} = new Answers()

direction('转 ','专砖')
news('拽专')
translate('hello world','注专转')
weather('驻转 转拽')
meaning('驻住驻')

```
